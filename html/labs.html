Hands-on Labs

This workshop includes hands-on labs covering bioinformatics tasks. Each lab builds on previous skills.

Running on Ibex

All labs are designed to run on the KAUST Ibex HPC cluster. Each lab includes the required module load command. Make sure you have an active Ibex account and can connect via SSH before the workshop. Please follow the instructions on the setup page to finish the Prerequisites.
